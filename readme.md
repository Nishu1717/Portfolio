# Portfolio Assets

This folder contains assets for your portfolio website:

## Files to Add:
- `resume.pdf` - Your resume/CV file
- `profile-photo.jpg` - Your professional headshot
- `project-screenshots/` - Screenshots of your projects

## Instructions:
1. Replace "Your Name" throughout the HTML with your actual name
2. Update email and social media links in the contact section
3. Add your actual project details and GitHub links
4. Customize the skills section based on your expertise
5. Add your education details in the about section
6. Upload your resume as `resume.pdf` in this folder
